# Campus Issue System

A full-stack web application designed to streamline how students report, track, and resolve campus-related infrastructure and administrative grievances. This platform bridges the communication gap between the student body and campus administrators for faster issue resolution.

---

## 🚀 Features

- **Student Dashboard:** Easy-to-use interface for students to file new complaints (e.g., hostel maintenance, Wi-Fi issues, classroom repairs) and view their live status.
- **Admin Control Panel:** Dedicated workspace for campus authorities to view incoming grievances, update progress status, and organize tasks.
- **Real-time Status Updates:** Instantly tracks whether an issue is *Pending*, *In Progress*, or *Resolved*.
- **Secure Authentication:** Managed user sign-ins to verify student and administration credentials securely.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Backend Environment:** Node.js, Express.js
- **Database & Backend Services:** Firebase (Authentication & Realtime Database/Firestore)
- **Project Configuration:** Webpack / npm package architecture
